# ⏱️ Shutdown Timer GUI

A lightweight, professional Windows utility built with **PowerShell** and **Batch** to schedule system shutdowns through a clean Graphical User Interface (GUI).

---

## ✨ Features
* **User-Friendly Interface:** Simple Windows Forms design for easy interaction.
* **Interactive Audio Feedback:** Distinct "beeps" notify you when a timer starts, cancels, or encounters an error.
* **No Installation Required:** Portable `.exe` file that works out of the box.

---

## 🚀 How to Use
1. Go to the **[Releases](https://github.com/A-Elmarakby/Shutdown-Timer-GUI/releases)** section.
2. Download the latest `Shutdown-Timer.exe`.
3. Run the application. 
   > **Note:** Since this is a custom tool, Windows might show a "SmartScreen" warning. Click **"More info"** and then **"Run anyway"**.
4. Enter the time in hours (e.g., `1` or `0.5`) and click **Start Timer**.
