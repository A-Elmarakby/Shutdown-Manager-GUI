# ⏱️ Shutdown Timer GUI - Auto Shutdown & Sleep Timer

A lightweight, native Windows utility built with **C# (Windows Forms)** to schedule system shutdowns easily. Perfect for users looking for a **Windows Schedule Shutdown**, **Auto Shutdown Timer**, or **Sleep Timer GUI** without messing with the command prompt.

---

## ✨ Features
* **User-Friendly Interface:** Simple Windows Forms design for easy interaction.
* **Interactive Audio Feedback:** Distinct "beeps" notify you when a timer starts, cancels, or encounters an error.
* **No Installation Required:** Portable `.exe` file that works out of the box.

---

## 🛡️ 100% Safe & Open Source
Because this is a standalone executable, some aggressive browser filters might initially flag it if it's newly downloaded. 
**This is a false positive.** The code is 100% open-source, and you can inspect the raw `ShutdownTimer.cs` file in the `src` folder yourself. No hidden scripts, no wrappers.

---

## 🚀 How to Use
1. Go to the **[Releases](https://github.com/A-Elmarakby/Shutdown-Timer-GUI/releases)** section.
2. Download the latest `Shutdown-Timer.exe`.
3. Run the application. 
   > **Note:** Since this is a custom tool, Windows might show a "SmartScreen" warning. Click **"More info"** and then **"Run anyway"**.
4. Enter the time in hours (e.g., `1` or `0.5`) and click **Start Timer**.

## ❓ FAQ (Frequently Asked Questions)
**How to schedule an automatic shutdown in Windows 10/11?**
Simply download this tool, enter the number of hours you want to wait, and click "Start Timer". No need to open CMD or type `shutdown -s -t`.

**How to set a sleep timer for PC without CMD?**
This GUI tool does exactly that. It runs natively on Windows and forces apps to close safely without requiring command-line knowledge.
